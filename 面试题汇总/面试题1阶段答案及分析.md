

# 第一题

=>	答案：undefined，undefined，12

```
1、变量声明提升，第一个打印=>undefined
2、函数执行开辟私有作用域，变量声明提升，形参赋值，第二个打印=>undefined
3、最后打印的是全局的a,所以结果是=>12
```

# 第二题

=>	答案：undefined，12 ，13

```
1、变量声明提升第一个=>undefined
2、函数执行及没有变量声明提升也没有形参赋值，所以a根据作用域链向上查找=>12
3、a = 13 ，对全局a重新赋值，最后结果=>13
```
